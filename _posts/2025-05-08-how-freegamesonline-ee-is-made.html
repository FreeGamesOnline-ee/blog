--- layout: post title: "How was FreeGamesOnline.ee amde?" date: 2025-05-08 categories: how --- FreeGamesOnline.ee uses Jekyll and GitHub Pages (hosting) to work properly. If you want to look at FreeGamesOnline.ee code, you can check it here! Even though Jekyll and GitHub Pages are the #1 used things on FreeGamesOnline.ee, there's still a few technologies used! Here's a list of them, from the most used to less used (from GitHub):
  1. HTML
  2. CSS
  3. JavaScript
  4. Python
  5. Shell
  6. PHP
There's a few things to consider for Python and Shell, we only use those to generate games pages (Python) and automatic Git commands like git pull or git push (Shell).
